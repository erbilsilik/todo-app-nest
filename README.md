## Installation

```bash
$ npm install
```

## Generate and serve compodoc documentation 

```bash
$ npx compodoc -p tsconfig.json -s
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run webpack
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
